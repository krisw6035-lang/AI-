# 微信小程序上线审核前检查清单

> 项目：房子魔法空间 · 新装  
> 版本：v0.4.0  
> 目标：确保所有项目在提交微信审核前已完成

## 一、小程序基础设置

- [ ] AppID 已配置为正式 AppID（非测试号）
- [ ] 小程序名称、图标、简介已填写完整
- [ ] 小程序类目已选择（建议：工具 > 图片处理 或 生活服务 > 装修装饰）
- [ ] 小程序头像已上传（建议 144x144px）

## 二、服务器域名配置

在微信公众平台 → 开发管理 → 开发设置 → 服务器域名：

- [ ] request 合法域名：`https://api.your-domain.com`（必须 HTTPS）
- [ ] uploadFile 合法域名：`https://api.your-domain.com`
- [ ] downloadFile 合法域名：`https://api.your-domain.com` + OSS 域名（如使用）
- [ ] socket 合法域名：无需配置（不使用 WebSocket）
- [ ] 域名已通过 ICP 备案

## 三、隐私与合规

- [ ] 用户协议页面内容完整，可从设置页访问
- [ ] 隐私政策页面内容完整，可从设置页访问
- [ ] 隐私政策已说明：收集 OpenID、上传图片、设计偏好、积分流水
- [ ] 隐私政策已说明：AI 服务由阿里云 DashScope 提供，图片会传输至该服务
- [ ] 隐私政策已说明：用户数据存储方式、用户删除数据的权利
- [ ] app.json 已配置 `__usePrivacyCheck__: true`
- [ ] app.js 已调用 `wx.requirePrivacyAuthorize()`
- [ ] 微信公众平台 → 用户隐私保护指引已配置

## 四、图片合规

- [ ] 上传图片仅限房屋/建筑/空间相关场景
- [ ] 图片大小限制已在前端和后端实现（≤10MB）
- [ ] 支持格式明确（jpg/jpeg/png/webp）
- [ ] 图片内容安全审核机制已准备（可接入微信内容安全 API 或自研）
- [ ] 用户协议的"用户义务"中已说明不得上传违规图片
