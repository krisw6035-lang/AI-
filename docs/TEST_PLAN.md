## 4. 核心路径验收（手动）

### 路径 1：室内改装

1. 打开小程序 → 首页显示
2. 点击「室内改装」→ 上传页
3. 选择照片 → 预览确认 → 下一步
4. 选择风格「现代极简」→ 选择肌理「天鹅绒」→ 选择色号「1391」
5. 勾选「沙发」「地毯」「绿植」
6. 输入自定义要求「希望客厅更明亮」
7. 点击生成 → 进入进度页
8. 轮询展示步骤 → 成功进入结果页
9. 查看原图/生成图对比 → 保存图片

### 路径 2：积分不足

1. 积分余额 < 10
2. 点击生成 → 提示「积分不足」

### 路径 3：未登录

1. Token 过期或不存在
2. 请求需鉴权接口 → 返回 401 → 触发重新登录
